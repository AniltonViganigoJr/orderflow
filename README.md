# OrderFlow

OrderFlow is an event-driven order management platform built with Spring Boot microservices, Apache Kafka, OpenFeign, PostgreSQL, and Docker.

## Architecture

Architecture diagram will be added as the project evolves.

## Tech Stack

- Java 21
- Spring Boot
- Kafka
- OpenFeign
- PostgreSQL
- Docker
